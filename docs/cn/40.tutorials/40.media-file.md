## Sample介绍
拍照、录像是无人机的重要功能，对拍摄的照片、视频等多媒体文件进行管理也就必不可少。多媒体文件的管理包括访问飞机存储空间内的多媒体文件资源、获取多媒体文件列表与列表状态、视频文件播放等。

本Sample会分别展示完整的接口以及接口调用流程的示例。
